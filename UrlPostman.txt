L'inscription se fait en 2 etapes
    1- http://localhost:8080/api/inscription [POST]:
    {
        "email" : "example@gmail.com",
        "mdp" : "mdpexample"
    }

    2- Coller l'url envoyee par email [GET]


Le login se fait en 2 etapes
    1-http://localhost:8080/api/login [POST]:
        {
            "email" : "example@gmail.com",
            "mdp" : "mdpexample",
            "type": "login"
        }

    2-http://localhost:8080/api/login [POST]:
        {
            "email" : "example@gmail.com",
            "pin": 00000,
            "type": "pin"
        }
